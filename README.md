## :checkered_flag: Starting ##

```bash

git clone https://github.com/FariasMi/beacademy-devstar-turmalina

# Access
cd beacademy-devstar-turmalina/projeto-empresarial

# install dependeces
composer install
# ------------------
npm i

# create a tables in your database

# run project

# ------------------
```
